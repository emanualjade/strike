# Example Skill

Use this template when adding a new production skill to `plugins/strike/skills/<skill-name>/`.

## Workflow

1. Replace `example-skill` with the lowercase kebab-case skill folder name.
2. Keep this file focused on the core agent workflow.
3. Move long examples or reference material into `references/`.
4. Put deterministic helper code in `scripts/`.
5. Put reusable output materials in `assets/`.
